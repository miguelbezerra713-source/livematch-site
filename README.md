# Landing page LiveMatch

Site estático independente do aplicativo principal.

## Publicar gratuitamente no GitHub Pages

1. Crie um repositório público novo no GitHub, por exemplo `livematch-site`.
2. Envie **somente o conteúdo desta pasta** para a raiz do novo repositório.
3. No GitHub, abra **Settings > Pages**.
4. Em **Build and deployment**, selecione **Deploy from a branch**.
5. Escolha a branch `main`, a pasta `/(root)` e clique em **Save**.

O endereço ficará disponível no formato `https://SEU-USUARIO.github.io/livematch-site/`.
